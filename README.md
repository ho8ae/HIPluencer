다양한 행사와 정보를 제공하는 힙합 커뮤니티 웹서비스 입니다.



## 1. 프로젝트 명
힙합 문화를 살리기 위한  다채로운 힙합 커뮤니케이션, HIPluncer

> 좋아하는 문화 중 하나인 '힙합' 이라는 문화를 다시 알리기 위해 제작되었습니다.
